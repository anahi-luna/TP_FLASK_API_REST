# 📦 TP Integrador - Sistema Fullstack "Libreta de Contactos"

Proyecto Fullstack desarrollado como trabajo práctico integrador, que consiste en una aplicación web para la gestión de contactos y localidades.

El sistema permite realizar operaciones CRUD completas, aplicar filtros dinámicos, validar datos tanto en frontend como backend y mantener relaciones entre entidades mediante una API REST.

---

# 🧠 Sobre el proyecto

La aplicación fue desarrollada utilizando:

- Frontend en React + Vite + TailwindCSS
- Backend en Flask + SQLAlchemy + Marshmallow
- Base de datos SQLite
- Contenerización con Docker y Docker Compose

Incluye validaciones, manejo de errores, diseño responsivo y arquitectura en capas para garantizar escalabilidad y buenas prácticas.
